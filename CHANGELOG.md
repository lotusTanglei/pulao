# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-09

### Added
- Initial release of Pulao: AI-Powered DevOps Assistant.
- **AI-Powered Deployment**: Deploy middleware using natural language commands.
- **System Ops**: Execute system shell commands safely via AI.
- **Multi-Provider Support**: Switch between OpenAI, DeepSeek, and other providers.
- **Interactive CLI**: Rich terminal interface with persistent header and command history.
- **Prompt Customization**: Configurable system prompts and clarification rules.
- **Gitee Support**: Optimized installation and update for users in China.
- **Cancellation**: Support for stopping AI generation with Ctrl+C.

### Changed
- Improved installation script to auto-detect region and select appropriate mirror (Gitee/GitHub).
