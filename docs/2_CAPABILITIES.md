# Pulao 当前能力矩阵 (Capability Matrix)

> 记录当前系统（v1.2.0）真实可用的能力。开发新功能后，必须同步更新此表。

## 1. 核心 AI 能力
- [x] **自然语言意图识别**: 能够理解模糊的运维指令。
- [x] **工具动态调用**: 基于 OpenAI Function Calling 标准。
- [x] **长期记忆 (RAG)**: 自动保存成功经验，并在相似场景下召回复用。
- [x] **LangGraph 编排**: 支持失败自动重试、多步工具连续调用。

## 2. 基础设施编排
- [x] **单机容器部署**: 自动生成 `docker-compose.yml` 并执行 `up -d`。
- [x] **集群多节点部署**: 通过 SSH 将配置分发至多个节点并分别拉起服务。
- [x] **模板匹配**: 支持读取内置/远程拉取的官方 Docker 模板作为生成参考。

## 3. 集群与节点管理
- [x] **集群生命周期**: 创建、切换 (`current_cluster`)、列表查询。
- [x] **节点管理**: 添加节点（自动验证 SSH）、移除节点、状态查询。

## 4. 缺失与不足 (待补全)
- [ ] **容器状态诊断**: 无法读取 `docker logs`，部署失败后 AI 只能瞎猜。
- [ ] **部署前环境预检**: 经常在部署执行后才发现端口被占用。
- [ ] **安全审计**: 缺乏镜像漏洞扫描和严格的细粒度权限控制。
